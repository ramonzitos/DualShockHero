DualShockHero
=============

DSHero is a program for playing GH-like(FOF, Phase Shift, GH3/4) programs with a
joystick, or, like the name says, with a DualShock.

emulator(PPJoy, vJoy).

2ยบ. Be sure that you have installed:
 - Python 2.7 (http://www.python.org/)
 - pygame module (http://www.pygame.org)
 - SendKeys module (http://www.rutherfurd.net/python/sendkeys/)

3ยบ. Run the DSConfig.py and see the instructions.

4ยบ. Open the program(DSHero/__init__.py)!
